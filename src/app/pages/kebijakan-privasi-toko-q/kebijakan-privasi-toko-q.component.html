Kebijakan Privasi Toko-Q

  • Info Aplikasi

    Aplikasi Toko-Q adalah aplikasi yang di rancang untuk mencatat transaksi penjualan bagi toko atau usaha yang ingin menggunakan aplikasi gratis.

  • Pengguna Aplikasi

    Target pengguna aplikasi Toko-Q adalah para pemilik toko sekala kecil yang membutuhkan aplikasi untuk mencatat penjualan seerta arus kas yang mudah digunakan dan tidak tergantung dengan internet.

  • Pertukaran Data

    Aplikasi Toko-Q tidak mengambil data apapun dari pengguna. semua data yang di input kedalam aplikasi hanya tersimpan di handpone masing-masing user tanpa ada syncronisasi dengan pihak manapun.
